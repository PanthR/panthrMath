Module panthrMath

Author(s): Haris Skiadas, Barb Wahl

Math computation library for PanthR

Provides special functions, probability distributions, and random number generators; inspired by the R code.

C

Frequently used constants. See C

Polynomial(coefs)

A small wrapper class for representing polynomial functions.

Polynomial.new(coefs)

Creates a polynomial function from coefs, an array of coefficients. For example, if the function is $ax^3 + bx^2 + cx + d$ the array is [a, b, c, d], that is, the array indexing is the reverse of the usual coefficient indexing.

Rational(num, denom)

A small wrapper class for representing rational functions.

Rational.new(num, denom)

Creates a rational function.

num and denom can be instances of Polynomial or arrays of coefficients for Polynomial.new.

Common factors between the numerator and denominator are not eliminated.

basicfunc

A collection of basic and special functions. See basicFunc.

distributions

Various probability distributions. See distributions.

Polynomial.prototype.evalAt(x)

Evaluates the polynomial at a given value x.

Rational.prototype.evalAt(x)

Evaluates the rational function at the value x.
